## Description

A minimal prompt to be used with the [min statusbar](https://gitlab.com/hthoreau/min_statusbar) for [tmux](https://tmux.github.io).

## Install

Either with omf

```fish
omf install nav
```

or [fisherman](https://github.com/fisherman/fisherman)

```fish
fisher gitlab.com/lusiadas/nav
```
